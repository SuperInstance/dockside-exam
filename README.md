Coast Guard certification checklist for fleet repositories.

* Automates Coast Guard certification checks for fleet repositories
* Provides pass/fail scoring across 7 key sections
* Streamlines compliance verification for fleet managers

### Quick Start
1. Clone the repository: `git clone https://github.com/SuperInstance/dockside-exam.git`
2. Run the exam: `./run-exam.sh`

### Architecture
The Dockside Exam is a simple, 7-section checklist with automated pass/fail scoring.
